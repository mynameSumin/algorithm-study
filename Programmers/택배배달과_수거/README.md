# 풀이 과정
### 1. 문제 정리
- n개의 집이 있다.
- 한 번에 cap개의 물건을 차에 실을 수 있다.
- 수거해야할 물건과 배달해야할 물건이 있을 때, 움직이는 최소 거리를 구해라.

### 2. 생각
- greedy 방식을 사용했지만 시간초과로 인해 1시간 내에 풀기 실패.
- 핵심은 greedy + 누적합을 사용하는 것이다.

### 3. 풀이 과정
다음과 같은 과정을 거쳤다.
-  가장 먼 집부터 들러 수거와 배달을 끝내야 한다. 따라서 가장 먼 집을 몇번 들러야하는지부터 계산한다.
- 들러야하는 횟수는 수거해야할 물건/cap, 배달해야할 물건/cap의 올림값 중에 큰 값이다.
- 만약 cap이 3, 들르는 횟수가 2, 수거해야할 물건이 4개라면 나는 다음 집에 갈 때 2개의 여유 공간이 있다. 이런 여유 값들을 dReamin, pReamin으로 관리한다.
