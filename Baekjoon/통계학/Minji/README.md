### 백준 2108 - 통계학
---------------------------------------

### 문제 설명
- n개의 수를 입력받아서 산술평균, 중앙값, 최빈값, 범위 구하기
- n은 홀수 개이다.

### 문제 해결
1. 산술 평균
- 각 수의 합이 양수일 때 : 합/개수
- 각 수의 합이 음수일 때
- 1) 합이 0이라면 return 0 (-0이 리턴되면 안됨)
- 2) 합이 0이 아닌 음수일 때 : 절댓값(합)/개수 한 후 -1을 곱해서 음수로 나타냄

2. 중앙값 : 오름차순 정렬 후 중앙값 반환

3. 최빈값
- 빈도수를 구하기 위해 unodered_map 사용 (속도 빠름)
- 각 빈도수를 맵에 매핑하고(1:2, 2:2 이런식으로 매핑됨) 벡터에 옮기기
- 내림차순 정렬 후 최빈값이 1개라면 index[0] 출력
- 최빈값이 2개 이상일 땐, index[1] 출력 (최빈값 중 두번째 값을 출력해야하므로)

4. 범위 : 오름차순 정렬 후 최댓값-최솟값을 한 후 절댓값을 씌어줌.

문제 링크 : https://www.acmicpc.net/problem/1940
